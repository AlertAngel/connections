# Connections

## TFT Screen


| TFT   | ESP32    |
|--------------- | --------------- |
| VCC  | 3v3   |
| GND   | GND   |
| CS   | GPIO09 (D9) |
| RESET   | GPIO08 (D8) |
| DC | GPIO2 (D2) |
| MOSI | GPIO06 (MO) (Pin number 24) |
| SCK | GPIO10 (SCK) (Pin number 23) |
| LED | 3v3 |


## I2C Sensors 

### MPU9250

| Sensor | ESP32 |
| -------------- | --------------- |
| VCC/Vin | 3v3 |
| GND | GND |
| SDA | GPIO21 |
| SCL | GPIO22 |


### MAX30102

| Sensor | ESP32 |
| -------------- | --------------- |
| VCC/Vin | 3v3 |
| GND | GND |
| SDA | GPIO21 |
| SCL | GPIO22 |


## OneWire Sensor

### DS18B20

| Sensor | ESP32 |
| -------------- | --------------- |
| VCC/Vin | 3v3 |
| GND | GND |
| S | GPIO27 |


## Buzzer 

| Buzzer | ESP32 |
| -------------- | --------------- |
| VCC/Vin | 3v3 |
| GND | GND |
| OUT | GPIO26 |
