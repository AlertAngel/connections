# Connections

## TFT Screen


| TFT   | ESP32    |
|--------------- | --------------- |
| VCC  | 3v3   |
| GND   | GND   |
| CS   | GPIO15 |
| RESET   | GPIO04 |
| DC | GPIO2 |
| MOSI | GPIO23 |
| SCK | GPIO18 |
| LED | 3v3 |

### MPU9250

| Sensor | ESP32 |
| -------------- | --------------- |

| GND | GND |
| SDA | GPIO21 |
| SCL | GPIO22 |


### MAX30102

| Sensor | ESP32 |
| -------------- | --------------- |
| VCC/Vin | 3v3 |

## OneWire Sensor

### DS18B20

| Sensor | ESP32 |
| -------------- | --------------- |
